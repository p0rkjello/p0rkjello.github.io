Remove ctrl-m characters on Linux

— Written by Andrew Bounds
#linux 

Use CTRL+V CTRL+M to type ^M

vi:

:%s/^M//g

tr:

tr -d ^M < file > newfile

sed:

sed 's/^M//g' file > newfile
